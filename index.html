materialshell

material color based dark theme
for your shell.


Github

Material Shell is a dark theme for your zsh-terminal based on Google's material design colors. Includes two versions of materialshell: Dark and Oceanic.

The color scheme supports OS X, Linux and Windows.

To install and use materialshell, read the documentation from the Github repository.
